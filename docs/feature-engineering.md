# Feature Engineering for Fraud Detection

## Velocity Features
- Transaction count per hour/day/week
- Unique merchants in last 24 hours
- Amount variance over rolling windows

## Behavioral Features
- Time since last transaction
- Distance from last transaction location
- Deviation from typical spending pattern

## Network Features
- Shared device fingerprints
- Common IP addresses across accounts
- Merchant risk scores

## Temporal Features
- Hour of day (cyclical encoding)
- Day of week
- Is holiday / weekend flag
